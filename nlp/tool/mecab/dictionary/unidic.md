

# [UniDic](http://www.tokuteicorpus.jp/dist/)
- 国語研短単位に基づく単位設計
- 可能性に基づく品詞体系
- 読み情報・発音情報だけでなくアクセント型などの情報を保持
- 固有名詞の標準形は「語形（かな）」が同じであれば認定
- Ver.2.0.1以降フリーライセンス(GPL/LGPL/BSD)になった
